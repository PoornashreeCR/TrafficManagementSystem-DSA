# TrafficManagementSystem-DSA
Traffic Management System using Graph Representation in Python

Description:

The Traffic Management System is designed to simulate and manage traffic intersections and road connections using graph data structures. The system models intersections as nodes and roads as edges using an adjacency matrix representation. Each intersection is associated with a traffic light that can be toggled between Red and Green states.

The system provides functionalities such as:
1.Initializing a road network
2.Adding roads between intersections
3.Toggling traffic lights
4.Displaying traffic light statuses
5.Displaying the road network structure

This implementation demonstrates practical usage of:
1.Graph Theory
2.Adjacency Matrix
3.Object-Oriented Programming
4.Menu-driven programs

Applications:
1.Smart city traffic control systems
2.Urban traffic simulation
3.Road network analysis
4.Academic learning of graph structures
